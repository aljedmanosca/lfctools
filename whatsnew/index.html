    What's New?

    This page will provide users with information on the latest and the greatest updates on LFCtools.

    New Features

    Leads-For-Full-review 5/16/2020

    prepare leads for full review using new template.

    Lead Searcher 5/13/2020

    ability to search leads from excel without the need to go through Kimako or Admin.
